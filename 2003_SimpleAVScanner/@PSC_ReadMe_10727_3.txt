Title: Simple AntiVirus Scanner
Description: The purpose of this program is to demonstrate how AntiVirus programs work at a very simple level. This program has two parts to it. The first part contains a detection writer which allows the user to create detections based on string and CRC values. The second part is to actually scan the file and report whether it is a threat or not.
